# Implementation of Stack, Queue, and Singly Linked List in Python #

Project Description
This project demonstrates the implementation of fundamental linear data structures— Stack, Queue, and Singly Linked List —using Python. It covers basic operations for each data structure and helps in understanding how data is stored, accessed, and manipulated efficiently in memory.

Stack Operations
- push(item)
- pop()
- peek()
- is_empty()
- display()

Queue Operations
- enqueue(item)
- dequeue()
- front()
- is_empty()
- display()

Singly Linked List Operations
- insert_at_beginning(data)
- insert_at_end(data)
- delete_node(data)
- search(data)
- count_nodes()
- find_middle()
- reverse_list()
- display()
